# employee-attrition-analysis
Questo progetto analizza l’abbandono dei dipendenti (employee attrition), ovvero la decisione di lasciare l’azienda. Attraverso tecniche di data science e machine learning, mira a individuare i fattori che influenzano questa scelta e a costruire modelli predittivi utili a supportare strategie di retention.

## Descrizione del progetto
Questo progetto analizza l’abbandono dei dipendenti (employee attrition), ovvero la decisione di lasciare l’azienda. Attraverso tecniche di data science e machine learning, si individuano i fattori che influenzano tale scelta.

## Dataset
Il dataset contiene informazioni sui dipendenti, tra cui:
- età, salario, ruolo
- soddisfazione lavorativa
- work-life balance
- anni di esperienza
- straordinari

La variabile target è:
**Attrition (Yes/No)**

## Obiettivo
- Analizzare i fattori che influenzano l'abbandono
- Individuare pattern nei dati
- Costruire modelli predittivi

## Modelli usati
(I vostri modelli - li completerete dopo)
- Logistic Regression
- Random Forest
- k-Nearest Neighbors
